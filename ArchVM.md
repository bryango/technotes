# Setting up an Arch VM

Many steps are similar to Manjaro{Install,Setup}.md.

- Download official qcow2 image
- Run in virt-manager (default NAT mode is okay with tailscale)
- Change hostname, add user, enable sshd
- **Install tailscale and enable tailscaled.service**
- Continue setup

## Essential packages
- tailscale
- neovim
- nix
